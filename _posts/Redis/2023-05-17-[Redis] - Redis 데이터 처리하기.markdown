## Redis 데이터 입력, 수정, 삭제, 조회하기:
### Redis 데이터 입력(저장)하기:
- 데이터를 저장할 때는 **set** 명령을 사용한다.
```bash
set key value
# 예시
ex) set data “hello”
```
