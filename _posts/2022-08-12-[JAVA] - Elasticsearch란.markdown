## Elasticsearch란? :
- Elasticsearch는 Apache Lucene(아파치 루씬) 기반의 java 오픈소스 분산 검색 엔진이다.
- 방대한 양의 데이터를 신속하고 거의 실시간으로 저장,검색,분석할 수 있다.
- 검색을 위해 단독으로 사용되기도 하며,  ELK( Elasticsearch / Logstatsh / Kibana )스택으로 사용되기도 한다.
