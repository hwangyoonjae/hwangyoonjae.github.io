## 로컬 브랜치(local branch) :
- 개발자가 자신의 컴퓨터에서 직접 작업하는 브랜치

```bash
# 브랜치 생성
$ git branch [브랜치명]

# 브랜치 변경
$ git checkout [브랜치명]

# 브랜치 목록 보기
$ git brach -r

# 생성한 local branch를 remote branch에 추가
$ git push origin [브랜치명]

# local branch, remote branch 연동
git branch --set-upstream-to origin/<브랜치 이름> 
```
